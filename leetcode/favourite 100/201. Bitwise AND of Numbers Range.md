# Question
Given a range [m, n] where 0 <= m <= n <= 2147483647, return the bitwise AND of all numbers in this range, inclusive.

Example 1:
    
    Input: [5,7]
    Output: 4
Example 2:

    Input: [0,1]
    Output: 0

# Solution
算法，代码如下：
```java
class Solution {
    public int rangeBitwiseAnd(int m, int n) {
        if(m == 0||n == 0)return 0;
        int res = 0;
        while(m != n){
            m = m >> 1;
            n = n >> 1;
            res++;
        }
        return m<<res;
    }
}
```
